埃博拉酱的图像处理工具包，提供一系列 MATLAB Image Processing Toolbox 内置函数所欠缺，但却常用的图像处理功能。

本包中所有函数均在ImageProcessing命名空间下，使用前需import。
```MATLAB
import ImageProcessing.*
```
函数文件内有详细文档，可用doc命令查看。
